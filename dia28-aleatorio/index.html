Nivel 1

  • onclick : Se activa al hacer clic con el ratón en el elemento.
  • onmousedown : Se activa cuando se presiona un botón del mouse sobre un elemento.
  • onmouseup : Se activa cuando se suelta un botón del ratón sobre un elemento.
  • onmousemove : Se activa cuando el puntero del mouse se mueve mientras está sobre un elemento.
  • onafterprint : Script que se ejecutará después de imprimir el documento
  • onbeforeprint : Script que se ejecutará antes de imprimir el documento
  • onload : Fires after the page is finished loading
  • onsubmit : Se activa cuando se envía un formulario
  • onmouseover : Se activa cuando el puntero del mouse se mueve sobre un elemento
  • ondrag : Script que se ejecutará cuando se arrastre un elemento
